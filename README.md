# Installation

**make sure your apt-get isn't broken before you run this**

**RUN THIS AS ROOT**
```bash
$ git clone https://github.com/Userphish/PhishX.git
$ cd PhishX 
$ chmod +x installer.sh
$ bash installer.sh
$ python3 PhishX.py
```
**Extra Info**
The [y/N] Questions are case sensitive so either you answer with "y" or with "N"
